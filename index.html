🌐 UMA SoC Interconnect

Featured

Apple M-Series style Unified Memory Architecture SoC Interconnect in synthesizable SystemVerilog. A 2x1 AXI4 Crossbar (axi4_crossbar) arbitrates between CPU (M0) and GPU (M1) bus masters using fixed-priority arbitration, routing AXI transactions by ID to a 256-bit wide Unified Memory Controller (axi4_uma_controller).

SystemVerilog AXI4 Protocol Yosys Synthesis NoC / Crossbar

🏎️ GPU Compute Core & RTCU

Featured

SIMT Streaming Multiprocessor (gpu_sm_core) managing 32-lane warps with a 256-register Vector Register File, integrated with a synthesizable Ray-Tracing Compute Unit (rtcu_core). The RTCU implements hardware FSM traversals (FETCH_BVH → INT_BOX → INT_TRI) and returns hit distances and barycentrics to the SM over a 256-bit memory bus.

SystemVerilog SIMT Architecture BVH Traversal Ray-Triangle ISA

💻 Bare-Metal Graphics Driver

Bare-metal C device driver interfacing a CPU application with the GPU Compute Core over a Unified Memory Interconnect. Implements a 32-byte aligned circular command queue (gpu_cmd_ring_t, 256 entries), MMIO register control at 0x80000000, asynchronous doorbell signaling, and zero-copy memory transfers via a shared UMA heap at 0x40000000.
